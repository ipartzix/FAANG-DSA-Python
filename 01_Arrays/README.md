# Arrays

## Key Concepts
- Contiguous memory storage
- Access by index: O(1)
- Insert/Delete/Search: O(n) worst-case
- Prefix sum / cumulative sum

## Common Patterns
1. Two pointers
2. Sliding window
3. Kadane's algorithm (max subarray)
4. Subarray sums and ranges

## ML Relevance
- Array preprocessing, feature vectors, sliding windows
