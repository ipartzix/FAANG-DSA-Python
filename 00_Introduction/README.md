# FAANG-DSA-Python

This repo is a structured collection of **FAANG-level DSA problems in Python**, aimed at:

- **Main goal:** Preparing for ML Engineer interviews
- **Backup goal:** Preparing for SDE interviews

## Structure
- Topics are organized by folder
- Each folder contains notes and starter Python code
- Problems are tagged by difficulty (Easy → Medium → Hard)

## Usage
- Start with easy problems, one folder at a time
- Focus on **patterns**, not just questions
- Gradually move to medium and hard FAANG problems
